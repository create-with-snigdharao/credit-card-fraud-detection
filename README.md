# Credit Card Fraud Detection

This project focuses on detecting fraudulent credit card transactions using machine learning.

## Objective
Build a classification model that predicts whether a transaction is fraudulent or legitimate.

## Tech Stack
- Python
- pandas
- numpy
- matplotlib
- scikit-learn
- imbalanced-learn
- Jupyter Notebook

## Project Structure
- `data/` dataset files
- `notebooks/` experiments and analysis
- `src/` reusable Python code
- `models/` saved models
- `reports/` plots and results

## Status
Project setup completed. Initial notebook creation in progress.
